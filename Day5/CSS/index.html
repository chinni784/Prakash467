The CSS Selectors

  • Element Selector
  • Id Selector
  • Class Selectors
  • Universal Selector
  • Grouping Selector

The Css element selector

All Html tags based on the element name is called as element selector

        Examples : p,h1,h3,img,bold,italic,etc..

        h1{
            text_align:center;
            color:red;
        }

The CSS Class selector

The class selector selects HTML element to with a specific class attribute.To select elements with a specific class,write a period(.),followed by the class naeme.

        Examples : HTML element with class="center"
    
        .center{
            text_align:center;
            color:red;
        }

The CSS Universal selector

The Universal selector(*) selects all the HTML elements on the page. 

        Examples : 
        *{
            text_align:center;
            color:red;
        }

The CSS Grouping Selector

To group selectors,seperate each selector with comma.

        Examples :
        h1,h2,p {
            text_align:center;
            color:red;
        }
